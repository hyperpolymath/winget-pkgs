== Bunsenite

=== Description

Bunsenite is a Nickel configuration file parser providing:

* **Rust core library** with stable C ABI (via Zig)
* **Multi-language FFI bindings**: Deno, ReScript, WebAssembly
* **Type safety** via Rust's type system
* **Memory safety** with zero unsafe blocks
* **Offline-first** operation (air-gapped compatible)

== Development Standards

This fork follows the **Hyperpolymath Standard** language policy (see `.claude/CLAUDE.md`):

* **Allowed**: ReScript, Deno, Rust, Tauri 2.0+, Dioxus, Gleam, Bash
* **Banned**: TypeScript, Node.js, Go, Python (except SaltStack)
* **Package Management**: Guix primary, Nix fallback, Deno for JS deps

== Workflow

=== Adding a New Package Version

[source,bash]
----
# 1. Sync fork with upstream
git fetch upstream
git merge upstream/main

# 2. Create manifest (use Microsoft's tool)
cd Tools
pwsh ./YamlCreate.ps1

# 3. Commit and push
git add manifests/h/Hyperpolymath/...
git commit -m "New version: Hyperpolymath.PackageName version X.Y.Z"
git push origin main

# 4. Create PR to microsoft/winget-pkgs
----

=== Manifest Location

----
manifests/
└── h/
    └── Hyperpolymath/
        └── PackageName/
            └── X.Y.Z/
                └── Hyperpolymath.PackageName.yaml
----
